Nedbank Namibia user journeys
=============================

Purpose
-------

Describe representative “I am the user” journeys for personal banking,
step-by-step, based on observed public information.

Journey A: Discover personal banking categories from homepage
-------------------------------------------------------------

Start
-----

- Homepage header navigation.

Steps
-----

- View the main navigation bar.
- Select the Personal segment tab (with Private, Business, Corporate, Group
  also visible as other segments).
- Hover over each Personal navigation bucket:
  - Bank
  - Borrow
  - Insure
  - Plan
  - Save & Invest
  - Help Centre

Outcome
-------

- Clear product taxonomy by need (Bank, Borrow, Insure, Plan, Save & Invest,
  Help Centre).
- Segment structure by customer type is visible (Personal, Private, Business,
  Corporate, Group).

Friction / clarity notes
------------------------

- Navigation is primarily hover-driven; behaviour on smaller screens or
  assistive technologies is not yet assessed.

Journey B: Find an everyday account and open the Silver PAYU details
--------------------------------------------------------------------

Start
-----

- Personal > Bank > Explore Bank.

Steps
-----

- From the Personal segment, open the Bank menu.
- Select Explore Bank.
- From the Explore Bank view, select the Accounts card.
- Within Accounts, locate the Everyday banking category.
- Within Everyday banking, open the Silver Pay-as-you-use account.

What is visible on the Silver PAYU page
---------------------------------------

- Product positioning as a low monthly fee, pay-as-you-use account.
- Monthly fee of N$30.
- Feature list describing capabilities such as overdraft access, payment
  options, and digital features.

Outcome
-------

- User reaches a detailed product page for Silver Pay-as-you-use, with
  headline pricing and feature information.

Friction / clarity notes
------------------------

- The separation between Everyday banking and other account types is clear,
  but comparative views across account options are not yet mapped.

Journey C: Attempt to apply for Silver PAYU
-------------------------------------------

Start
-----

- Silver Pay-as-you-use product page.

Steps
-----

- Read the “How to apply” or equivalent section.
- Review eligibility and basic requirements, including:
  - Namibian residency.
  - Minimum and maximum income band.
  - Age requirement (18+).
  - KYC documentation expectations.
- Locate the “Ways to open” information.
- Review available onboarding channels:
  - Web “leave a message” request.
  - Facebook IdentityToday selfie sign-up path.
  - Branch.
  - Call centre.

Outcome
-------

- User learns that there are multiple channels for onboarding, including
  remote and in-person options.

Friction / clarity notes
------------------------

- Exact digital onboarding flow steps and decision points are not yet mapped.
- Dependencies on third-party identity systems (for example IdentityToday)
  are visible but not yet unpacked.

Journey D: Request insurance information
----------------------------------------

Start
-----

- Personal > Insure.

Steps
-----

- From the Personal segment, open the Insure menu.
- Select an insurance category (for example short-term insurance, life cover,
  credit life insurance, or funeral cover).
- Open a specific insurance product page.
- Click a call-to-action such as “Get a callback”.
- View and complete the callback form fields, including:
  - Name and surname.
  - Contact number.
  - Email address.
  - Request type (insurance or planning).
  - Insurance subtype options.
  - Planning subtype options.
  - Preferred callback time windows.
- Submit the request.

Outcome
-------

- User requests follow-up contact for insurance or planning, with basic
  information captured for an outbound call or advisory interaction.

Friction / clarity notes
------------------------

- Disclaimer or legal links may loop back to the same page, which can reduce
  clarity about the underlying terms being accepted.
