Namclear data layer map
=======================

Objective
---------

Outline the technical and data architecture around Namclear-operated schemes,
as far as can be understood from public information.

Suggested views
---------------

- Core processing and clearing systems
- Interfaces with participant banks and other institutions
- Settlement processes and links to central bank money
- Monitoring, risk, and reporting systems

For each component or flow, capture:

- Data structures or message types
- Direction of flows
- Frequency and settlement cycles
- Dependencies on standards and external infrastructures
