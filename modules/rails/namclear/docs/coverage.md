Namclear coverage
=================

Purpose
-------

Provide a concise view of how complete the Namclear analysis is across
different dimensions.

Dimensions
----------

- Scheme and infrastructure overview
- Participants and access models
- User and participant journeys
- Systems and data
- Risks and controls
- Opportunities and hypotheses

Assessment
----------

For each dimension, provide:

- A qualitative rating (for example: Initial, Partial, Good, Deep)
- Key gaps and assumptions
- Planned next steps
