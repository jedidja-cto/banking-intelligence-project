# Data policy

This project uses synthetic data only.

## Allowed
- Synthetic datasets generated locally
- Publicly available information such as product descriptions, fee tables, and eligibility requirements

## Not allowed
- Any real customer data
- Any private bank data
- Any scraping that violates terms or bypasses access controls

## Files
- `synthetic/customers_sample.csv` Small sample customer table
- `synthetic/transactions_sample.csv` Small sample transaction table
