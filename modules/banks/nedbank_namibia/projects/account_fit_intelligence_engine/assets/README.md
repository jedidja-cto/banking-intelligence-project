## File Naming Convention

Use descriptive names with dates:
- `architecture_diagram_2024-01-20.png`
- `stakeholder_presentation_v0.1.pdf`
- `fee_comparison_chart.png`

## Source Attribution

All materials must include source attribution:
- Public documents: Cite source URL and date accessed
- Generated visualizations: Note data source and generation date
- Third-party materials: Include proper licensing information
